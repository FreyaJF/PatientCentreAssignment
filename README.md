## Overview

A take-home assignment for The Patient Centre. A simple system for tracking events, buffering them, and sending them to a mock API. Includes unit tests.

## Requirements

Tested on Python 3.12.

## Project Structure

├── AnalyticsBuffer.py 

Analytics buffer implementation
  
├── MockAPI.py 

Mock API implementation
  
├── test_analytics_buffer.py 

Automated unit tests
  
└── README.md

## How to run the unit tests

1) Clone the repository:

git clone https://github.com/FreyaJF/PatientCentreAssignment.git

2) Navigate to the new directory and run the unit tests using:

python -m unittest test_AnalyticsBuffer.py
